Модальные окна

.modal \ .modal--search

modal-search использует компонент +search-form(), он обязательно должен присутствовать на странице.

На разрешении больше 1280px отображается компонент +search-form(), а кнопка вызова модального окна блокируется.

На разрешении меньше 1280px компонент +search-form() переносится в модальное окно, кнопка вызова модального окна становится активной.

+btn().btn--search(aria-pressed="false", aria-label="Переключатель отображения поиска", data-open-modal="search")
  +icon("icon-search", "20", "20")

Обертка для заголовка с текстом

Обертка для заголовка с текстом, модификатор --closer имеет меньший margin-bottom у пследнего дочернего элемента этого компонента.

Модификатор --no-margin-bottom убирает margin-bottom у последнего дочернего элемента. Нужно когда у следующего компонента есть свой внутренний вертикальный отсуп сверху.

.page-title оборачивает заголовок и текст под ним, если он имеется.

.page-title \ .page-title--closer \ .page-title--no-margin-bottom

Баннер

В миксин передает блок с контентом из одного или нескольких параграфов, ссылки должны быть обернуты в тег ссылки. Теги a и p стилизованы от обертки с классом .banner

.banner

Места за столиками бронируйте заранее по телефону +7 (495) 532-47-23, WhatsApp/Viber +7 (926) 258-99-20

+banner({name: "icon-star", width: 46, height: 46})
  p Места за столиками бронируйте заранее по телефону #[a(href="tel:+74955324723") +7 (495) 532-47-23], WhatsApp/Viber #[a(href="tel:+79262589920") +7 (926) 258-99-20]

В компонент передаются объекты по колонкам dataLeft и dataRight. Логотип

Логотип будет обернут в span, если страница index.html

Если страница не index.html или передать флаг isFooter, то логотип будет обернут в ссылку

Если добавить флаг isFooter, на разрешении меньше 768px логотип скрывается

Аргументы currentUrl и isFooter передавать не обязательно

.logo \ обернут в span, т.к. currentUrl === "index.html"

+logo({name: "logo", width: "161", height: "72", currentUrl: "index.html"})

.logo \ обернут в link, т.к. currentUrl !== "index.html"

+logo({name: "logo", width: "161", height: "72", currentUrl: "about.html"})

.logo \ обернут в link, т.к. isFooter: true

+logo({name: "logo", width: "161", height: "72", currentUrl: "index.html", isFooter: true})

.logo \ .logo--footer

+logo({name: "logo", width: "161", height: "72", isFooter: true}).logo--footer
